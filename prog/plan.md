# 목적

1. **시장 타이밍**: 지금이 투자(매수)하기 좋은 타이밍인가, 관망/매도할 때인가 알려준다.
2. **종목 타이밍**: 개별 종목의 매수/매도 타이밍을 알려준다.

투자 대상: **미국 주식** (현재 시장 verdict 구현 범위), **한국 주식** (개별 종목 분석만), ETF.

---

# 운영 원칙

- **조회 시점**: 사용자가 명령을 친 시점에 실시간 조회. 캐시 없음.
- **해상도**: 일봉 기준. 장중 틱 신호는 다루지 않음.
- **속도**: 명령 한 번에 수십 초 걸려도 감수.
- **결과 톤**: "지금 매수/관망/매도 구간" 수준의 단순한 권고.
- **외부 키 불필요**: FRED 는 공개 CSV 엔드포인트 사용, CNN F&G 는 공개 JSON.

# 투자 스타일과 모드

- **기본 모드**: 중장기 (몇 달~몇 년 보유).
- **예외**: 시장 상황에 따라 스윙(며칠~몇 주)으로 전환.
- **모드는 목적 1(시장 타이밍) 결과가 결정**하고, 그 모드가 목적 2(종목 타이밍)의 블록 가중치를 바꾼다.

### 모드 매핑

| 시장 총점 | 판정 | 모드 | 종목 판단 강조 |
|---|---|---|---|
| ≥ +0.4 | 🟢 매수 우위 | 중장기 | 밸류·퀄리티 45% · 이벤트 30% · 추세 15% · 섹터 10% |
| −0.4 ~ +0.4 | 🟡 혼조 | 스윙 | 추세 45% · 이벤트 25% · 섹터 15% · 밸류 15% |
| ≤ −0.4 | 🔴 하락장 | 관망 | 매수 신호 1단계 하향, "모드 불일치 경고" 표시 |

# 목적 1의 판단 항목 (미국 주식)

6개 코어 수치 지표. 각 지표는 -1/0/+1 로 매핑 후 가중 합산 → 총점.

| 코드 | 지표 | 가중 | 원천 | 룰 |
|---|---|---:|---|---|
| **C1_BS**  | Fed 대차대조표 4주 Δ | 0.10 | FRED WALCL | ≥+$50B QE / ≤−$50B QT |
| **C1_TGA** | TGA 잠재 + 방향 | 0.10 | FRED WTREGEN | 잔량 ≥$700B(+) · ≤$200B(−) × 4주 감소(+) · 증가(−) |
| **C1_RRP** | RRP 잠재 + 방향 | 0.10 | FRED RRPONTSYD | 잔량 ≥$500B(+) · ≤$50B(−) × 4주 감소(+) · 증가(−) |
| **C2** | S&P500 vs 200일선 | 0.30 | yfinance `^GSPC` | ±0.5% 데드존 |
| **C3** | HY OAS 크레딧 스프레드 | 0.25 | FRED BAMLH0A0HYM2 | <400bp 안정 / >600bp 경계 |
| **C4** | 섹터 로테이션 (공격−방어) | 0.15 | yfinance 섹터 ETF | ±1.0%p (Tech·Discr·Indus vs Util·Staples·Health) |

**유동성 철학 (C1\_\*)**: 단순히 "지금 흐르는 돈"이 아니라 **"앞으로 풀릴 수 있는 연료"(잠재) + "지금 어느 방향으로 움직이나"(4주 변화)** 를 합쳐 판정. 두 시그널이 같은 방향이면 ±1, 엇갈리면 0.

**맥락 패널(판정 기여 X, 참고용)**: CNN Fear & Greed, 10Y UST, 10Y−2Y, HY OAS, Fed BS, RRP, TGA. 레거시 가격 패널 (KOSPI, NASDAQ, USD/KRW, 10Y UST).

**판정 결정**:
- 총점 ≥ +0.4 → 🟢 매수 우위 → 중장기
- −0.4 ~ +0.4 → 🟡 혼조 → 스윙
- 총점 ≤ −0.4 → 🔴 하락장 → 관망

**누락 처리**: 공급된 지표만의 가중 합으로 정규화 (분모 자동 축소). 전부 누락이면 총점 0 → MIXED.

# 목적 2의 판단 항목 (종목)

4개 블록. 모드별 가중치로 조합.

| 코드 | 블록 | 서브 지표 |
|---|---|---|
| **TREND**  | 추세·가격 | 주가/200일선, RSI(14), MACD, SMA 크로스, 거래량(5/20) |
| **VALUE**  | 밸류·퀄리티 | PER(vs forwardPE 기준), ROE, 부채비율, 매출 성장 YoY |
| **EVENT**  | 이벤트 (미구현) | 실적·공시, 어닝 리비전, 내부자 거래 |
| **SECTOR** | 섹터 맥락 | 전체 섹터 1개월 평균 모멘텀 |

모드별 가중치:
- 중장기: VALUE 0.45 · EVENT 0.30 · TREND 0.15 · SECTOR 0.10
- 스윙: TREND 0.45 · EVENT 0.25 · SECTOR 0.15 · VALUE 0.15
- 관망: 균등 0.25 (매수 신호 1단계 하향 + 경고 표시)

블록 점수 = 서브 지표들의 `-1/0/+1` 평균. 총점 = 블록 점수 × 모드 가중치 합.

**5단계 신호 매핑**: ≥+0.6 STRONG BUY / ≥+0.2 BUY / -0.2~+0.2 HOLD / ≤-0.2 SELL / ≤-0.6 STRONG SELL.

# 결과 형식

**원칙**: 각 지표마다 현재값·관찰·기준 규칙을 모두 보여준다. 🟢🔴 만 찍지 않는다.

## 시장 타이밍 출력 (실제 포맷)

```
[코어] 미국 주식 시장
🟢 매수 우위   총점 +0.70   모드 중장기

🟡 Fed 대차대조표    6.71T$
   4주 Δ +50B  [주간]   →  중립  기여 +0.00
   기준: 4주 ≥+50B QE / ≤-50B QT

🟢 TGA 잠재+방향    0.75T$
   0.75T$ · 4주 Δ -102B  [주간]   →  우호  기여 +0.10
   기준: 잔량 ≥700B(+) · ≤200B(-) / 4주 감소(+) · 증가(-)

🟢 S&P500 / 200일선    7,118
   +6.4% (200일선 6688)  [일일]   →  우호  기여 +0.30
   기준: ±0.5% 데드존 (위/아래)

...
```

3줄/지표: (1) 아이콘 + 이름 + 현재값 (2) 관찰 + 해상도 → 판정 + 기여 (3) 기준 규칙. 80자 터미널에서 줄바꿈 0.

누락 지표 있으면 하단에 "누락: XXX (분모 자동 정규화)" + FRED 필요 지표면 네트워크 안내.

## 종목 타이밍 출력 (실제 포맷)

```
Apple Inc. (AAPL)
🟢 BUY   총점 +0.24   모드 중장기

▎TREND · 추세·가격  블록 점수 +0.20   기여 +0.03
  🟢 주가/200일선  +8.3% (200일선 252.45)
     기준: ±3% 데드존
  🟡 RSI(14)  RSI 66.9
     기준: ≤30 과매도 +1 / ≥70 과매수 -1
  ...

▎VALUE · 밸류·퀄리티  블록 점수 +0.25   기여 +0.11
  🟡 PER  34.7 (기준 29.2)
     기준: 기준 ×0.80 이하 저평가 / ×1.30 이상 고평가
  🟢 ROE  152.0%
     기준: ≥15% 우수 / ≤8% 경계
  ...

▎EVENT · 이벤트·엣지  블록 점수 +0.00   기여 +0.00
  데이터 없음

▎SECTOR · 섹터 맥락  블록 점수 +1.00   기여 +0.10
  🟢 섹터 모멘텀(평균)  전체 섹터 1개월 평균 +6.1%
```

# 인터페이스 (CLI)

현재 구현:

| 명령 | 역할 |
|---|---|
| `ivst market` | 목적 1 수행 — US 시장 verdict + 맥락(F&G, FRED) + 가격 패널 |
| `ivst signal [TICKER]` | 목적 2 수행. 인자 없으면 관심종목 전체 요약, 있으면 단일 종목 상세 |
| `ivst watch add/list/remove` | 관심종목 관리 |
| `ivst portfolio add/list/perf` | 포트폴리오 관리 + 수익률 |
| `ivst note add/list/search` | 리서치 노트 |
| `ivst news [--all]` | 뉴스 감성 태깅 (키워드 기반) |
| `ivst recommend [--reason --market]` | 섹터 모멘텀·정책 기반 추천 |
| `ivst dashboard` / `ivst` | 통합 뷰 (market + signals + news) |

# 범위 결정 기록 (스코프 축소 이력)

진화 과정에서 다음 항목들을 **축소 또는 제거**:

- **금/은**: DXY·실질금리·PMI 기반 verdict 제거. 투자 대상에서 제외.
- **한국 시장 verdict**: pykrx 의 외국인 순매수 엔드포인트가 KRX 로그인 요구로 막힘. KOSPI/200DMA 와 원/달러는 기술적으로는 가능했으나 "시장 집계"는 제거. **개별 한국 종목 분석(pykrx OHLCV)은 유지**.
- **M2**: 월간·지연이라 일봉 타이밍 부적합. 제외.
- **VIX**: CNN Fear & Greed 로 교체 (F&G 는 VIX 를 구성요소로 포함하는 상위 복합 지표).
- **FRED API 키**: 공개 CSV 엔드포인트로 전환하여 불필요.
- **LLM 맥락 레이어**: Phase F 로 연기. 현재 뉴스 감성은 keyword 기반 `sentiment.py` 가 `news` 커맨드에서만 사용.
- **자금시장 스트레스(SOFR−IORB)**: 미구현. 필요 시 추가.
- **어닝 리비전(EVENT 블록)**: yfinance 스키마 불안정으로 연기. 현재는 "데이터 없음".
