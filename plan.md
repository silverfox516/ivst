# 목적

1. **시장 타이밍**: 지금이 투자(매수)하기 좋은 타이밍인가, 관망/매도할 때인가 알려준다.
2. **종목 타이밍**: 개별 종목의 매수/매도 타이밍을 알려준다.

투자 대상: 미국 주식, 한국 주식, ETF.

---

# 운영 원칙

- **조회 시점**: 사용자가 명령을 친 시점에 실시간 조회. 캐시 없음.
- **해상도**: 일봉 기준. 장중 틱 신호는 다루지 않음.
- **속도**: 명령 한 번에 수십 초 걸려도 감수.
- **결과 톤**: "지금 매수/관망/매도 구간" 수준의 단순한 권고.

---

# 투자 스타일과 모드

- **기본 모드**: 중장기 (몇 달~몇 년 보유).
- **예외**: 시장 상황·정책에 따라 스윙(며칠~몇 주)으로 전환. 시장을 거스르지 않는다.
- **모드는 목적 1(시장 타이밍) 결과가 결정**하고, 그 모드가 목적 2(종목 타이밍)의 강조점을 바꾼다.

### 모드 매핑

| 시장 타이밍 결과 | 모드 | 종목 판단 강조 |
|---|---|---|
| 🟢 매수 우위 · 상승장 · 정책 우호 | 중장기 | 이평선, PER/PBR, ROE, 재무 |
| 🟡 혼조 · 고변동성 · 단기 악재 | 스윙 | RSI, 단기 지지선, 거래량 |
| 🔴 하락장 · 정책 악재 · 공포 | 관망 | 신규 진입 자제, 기존 보유만 관리 |

---

# 목적 1의 판단 항목

6개 블록. 1~4·6은 수치 지표, 5는 LLM 질적 해석.

| # | 블록 | 지표 | 원천 |
|---|---|---|---|
| 1 | **유동성** | Fed 순유동성(Fed BS − TGA − RRP, 일일), RRP 잔액 추세, Fed 대차대조표(주간 H.4.1) | FRED, NY Fed |
| 1b | 자금시장 스트레스 *(보조)* | SOFR − IORB 스프레드 (평소 0~5bp, 스파이크 시 경색 경보) | NY Fed |
| 2 | **금리·통화정책** | Fed 스탠스(FOMC 성명·점도표·파월 발언), 한국은행 금통위, 10Y−2Y 스프레드(침체 선행) | FOMC, BoK, FRED |
| 3 | **추세·폭** | 주가지수 vs 200일선(S&P500, KOSPI), 200일선 위 종목 비율(breadth) | 시세 API |
| 4 | **리스크·심리** | VIX, HY 크레딧 스프레드, (보조) Fear&Greed | FRED |
| 5 | **뉴스·정책 맥락 (LLM)** | 재정 방향, 무역·관세, 정치 리스크, 뉴스 감성(7일) | 주요 매체 RSS + LLM |
| 6 | **섹터 로테이션** | 섹터 ETF 수익률 랭킹 (공격 vs 방어) | 시세 API |

- M2는 월간·지연이라 제외 (일봉 타이밍 부적합).
- 5번은 뉴스·정책을 LLM이 요약해 양/음 성향만 판정. 세부 항목별 임계치를 두지 않는다.
- 6번의 섹터 결과는 목적 2에서도 재사용한다.

# 목적 2의 판단 항목

4개 블록.

| # | 블록 | 지표 |
|---|---|---|
| 1 | **추세·가격** (차트) | 주가/200일선, RSI(14), MACD, 거래량(5·20일) |
| 2 | **밸류·퀄리티** (재무) | PER, PBR, ROE, 부채비율, 매출·이익 성장 |
| 3 | **이벤트** (개별) | 실적·공시, 기업별 규제·소송, 내부자 거래 |
| 4 | **섹터 맥락** | 목적 1의 섹터 로테이션 결과 재사용 (가격 + 정책 양축) |

# 결과 형식

**원칙**: 기준을 명시하고, 현재값과 비교해 판정을 보여준다. 추상적 신호등만 쓰지 않는다.

## 시장 타이밍 출력 예시

```
시장 타이밍: 🟢 매수 우위 (모드: 중장기)

항목              현재값     기준                          판정
──────────────────────────────────────────────────────────────
유동성 (M2 YoY)    +4.2%    >+2% 확장 / <0% 긴축          🟢 확장
역레포 잔액        $0.4T    감소 추세(3개월) 시 우호       🟢 우호
FOMC 스탠스        동결     인하/동결 우호 / 인상 경계     🟡 중립
VIX                18       <20 안정 / >30 공포            🟢 안정
뉴스 감성(7일)     -0.3     >0 긍정 / <-0.2 부정           🔴 부정
섹터 모멘텀 1위    기술     방어주 주도 시 경계            🟢 공격적

요약: 유동성·모멘텀 우호. 관세 뉴스로 단기 변동성 확대
가능 → 신규 진입은 분할 매수 권고.
```

## 종목 타이밍 출력 예시 (모드: 중장기)

```
AAPL 종목 타이밍 (모드: 중장기)

▶ 중장기 [강조]
항목            현재값   기준                           판정
──────────────────────────────────────────────────────────
주가/200일선    +3.2%    위/아래                        🟢 위
PER             28       업종평균 22 대비               🟡 다소 높음
ROE             26%      >15% 우수 / <8% 경계           🟢 우수
부채비율        35%      <50% 안전                      🟢 안전

▷ 스윙 [참고]
RSI(14)         55       <30 과매도 / >70 과매수        🟡 중립
단기 지지선     $168     근접 시 반등 후보              ─
거래량(5일)     +10%     +30% 이상 매집 추정            🟡 중립

요약: 펀더멘털 양호, 추세 유지 중. PER 부담은 있으나
중장기 보유 적합.
```

- **모드 전환 시 순서 반전**: 스윙 모드면 스윙 섹션이 상단·강조, 중장기는 참고.
- **관망 모드**에서는 매수 신호가 있어도 "모드 불일치 경고"를 붙인다.
- **판정 기준은 설정 파일(예: `config/thresholds.yaml`)로 분리**하여 사용자가 수정 가능하게 한다.
