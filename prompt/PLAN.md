# 투자 보조 프롬프트 제작 계획

## Context

국내·미국 주식(ETF 제외) 매수·매도 판단을 보조할 AI 챗(웹 검색 가능 환경) 프롬프트 2종을 만든다.
- 종목 분석 1종
- 시장·거시·유동성 분석 1종

합의된 결정 사항:
- 대상 AI: 웹 검색 가능 모델 (ChatGPT/Claude/Gemini 무관)
- 시간축: 스윙(수주~수개월)과 중장기(1년+) 둘 다, 분리 출력
- 출력 깊이: 결론 먼저(3~5줄) → 본문 표 중심(A4 1.5장) → 마지막 `## 한 줄 결론` 강제
- 출력 언어: 한국어 (티커·숫자·영문 고유명사는 원문 유지)
- 옵션: 시간축 분리 출력, 출처+조회날짜 명시, 불확실성("미확인") 표기 강제

## 산출물

| 파일 | 용도 |
|---|---|
| `prompts/stock-analysis.md` | 종목명·티커 입력 → 종목 분석 출력 |
| `prompts/macro-liquidity.md` | 시점 입력 → 시장·거시·유동성 분석 출력 |
| `README.md` | 사용법, 입력 예시, 한계, 면책, 갱신 주기 |

## 공통 설계 원칙

### 출력 구조
- **헤더 결론**(맨 위 3~5줄): 한 줄 결론 + 매수/매도/관망(또는 시장 국면) + 핵심 가격대
- **본문**: 표 중심 + 불릿, A4 1.5장 분량
- **`## 한 줄 결론`**(맨 마지막, 강제): 의견 + 가격대/국면 + 시간축 명시

### 데이터 규칙
- 모든 수치에 **출처를 클릭 가능한 마크다운 링크**로 표기 — `[출처명](URL)`, 가능하면 해당 지표·문서를 바로 여는 deep link
- 모든 수치에 **조회 날짜** 표기
- 모르거나 데이터가 오래된 경우 **"미확인"** 명시 → 환각 방지
- 본문 한국어, 티커·숫자·영문 고유명사는 원문 유지

### 시간축
- 스윙(수주~수개월)과 중장기(1년+) 의견 **분리 출력**
- 두 시간축이 상충하면 그 이유 명시

## 프롬프트 A: 종목 분석

### 입력
- 종목명 또는 티커 단독 (예: `NVDA`, `삼성전자`)
- 부가 정보(매수가·보유 여부·손익) 받지 않음 → **요청 시점 기준 전망**만

### 출력 섹션 (우선순위 반영)
종목 고유 분석보다 **섹터·업계 흐름·정책 수혜**를 먼저 다룬다.

1. **헤더 결론** — 종목명/티커, 현재가, 매수/매도/관망, 한 줄 근거 (섹터·정책 코멘트 포함)
2. **섹터·업계 흐름 (우선)** — 섹터 수익률, 업계 사이클 위치, 업계 분위기
3. **경쟁업체 비교우위 (우선)** — 핵심 경쟁사 표 비교 + 항목별 평가(기술·원가·브랜드·생산·자본) + 상대적 강·약점 + 점유율 추세
4. **정책·규제 수혜·리스크 (우선)** — 한국 정부 정책·한은, 미국 행정명령·법안·연준·FTC, 정량적 영향 추정
5. **비즈니스 요약** — 수익 모델, 매출 구성, 자체 해자
6. **재무 스냅샷 (표)** — 3~5년 매출/영업이익/순이익/FCF/부채비율
7. **밸류에이션 (표)** — PER/PBR/PSR/EV·EBITDA + 동종업계·자체 5년 평균 비교
8. **모멘텀·수급** — 추세, 외국인·기관 수급, 옵션 IV·숏 인터레스트, 이평선
9. **촉매·리스크** — 다가오는 실적·이벤트, 거시 민감도, 종목 고유 리스크
10. **시간축별 의견** — 스윙(진입/손절/목표가), 중장기(적정가 + 보유/추가매수/축소)
11. **`## 한 줄 결론`**

## 프롬프트 B: 시장·거시·유동성

### 입력
- 시점 키워드 (`오늘`, `이번주`, `2026-04-29`)

### 출력 섹션
사용자가 매크로·유동성에 익숙하지 않으므로 **모든 지표에 한 줄 설명 + 학습용 링크** 동반.

1. **헤더 결론** — 시장 국면 + 한 줄 근거
2. **매크로 (표)** — 미국(Fed Funds, 10Y/2Y, CPI, Core PCE, NFP, ISM), 한국(기준금리, CPI, 수출, USDKRW)
3. **유동성 (표)** — 미국(Fed B/S, RRP, M2, DXY), 한국(한은 RP, 외국인 누적 순매수)
4. **시장 내부 지표** — VIX, MOVE, HY 스프레드, 풋콜비, AAII
5. **섹터 로테이션** — 1주·1개월 섹터 수익률, 주도/소외 섹터
6. **시장 국면 판단** — 종합 국면, 직전 대비 변화
7. **포지셔닝 가이드 (시간축별)** — 스윙(비중·헤지), 중장기(자산배분)
8. **주요 일정** — 1~2주 내 FOMC/CPI/실적
9. **`## 한 줄 결론`**

### 표 컬럼 규격 (매크로·유동성·시장 내부·섹터 로테이션 공통)
| 컬럼 | 내용 |
|---|---|
| 지표명 | 한글명 (영문 약어) |
| 한 줄 설명 | "왜 보는가" 한 줄 |
| 현재값 | 수치 + 단위 |
| 직전 | 전월·전주 대비 변화 (화살표) |
| 의미 | 위험선호 / 중립 / 위험회피 |
| 출처 | 클릭 가능한 데이터 원본 링크 + 조회 날짜 |
| 더 알아보기 | 한국어 우선 학습 링크 (Investopedia, 한은 경제용어사전, FRED 시리즈, 위키 등) |

## 한계·면책

- AI 출력은 투자 판단의 **보조**, 최종 책임은 사용자
- 실시간 호가·체결은 부정확 가능 → 매매 직전 HTS/MTS 재확인
- 거시 지표는 발표 시차 존재 → 조회 날짜 반드시 확인
- 프롬프트는 **분기 1회** 검토·갱신
